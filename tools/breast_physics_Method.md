# 📐 MMD/3D模型胸部顶点组拟合指南：球心与半径计算

本文档旨在提供最准确的算法，用于通过顶点组（Vertex Group）计算女性模型胸部的近似球心（Sphere Center）和半径（Radius）。该方法基于**加权平均**原理，适用于 MMD 刚体生成、碰撞体包裹及动态物理模拟。

## 🎯 核心原理：为什么加权平均最准确？

在 3D 蒙皮模型中，顶点并非同等重要。
*   **普通几何中心**：容易被胸部边缘稀疏的布线或噪点顶点拉偏。
*   **加权质心**：离骨骼越近、权重（Weight）越大的顶点，对球心和半径的“话语权”越大。这符合物理规律（质量中心），也能让计算出的球心始终紧贴骨骼运动的核心区域。

## 🛠️ 计算步骤详解

### 第一步：数据提取与筛选
从模型网格中提取属于目标胸部顶点组的所有数据。
*   **顶点坐标集合**：$P = \{p_1, p_2, ..., p_N\}$ （$p$ 为 XYZ 坐标向量）
*   **对应权重集合**：$W = \{w_1, w_2, ..., w_N\}$ （$w$ 为该顶点对胸部骨骼的蒙皮权重，范围 0~1）

### 第二步：计算近似球心（加权质心）
球心 $C$ 是所有顶点坐标的加权平均值。

**数学公式：**
$$C_{center} = \frac{\sum_{i=1}^{N} (p_i \times w_i)}{\sum_{i=1}^{N} w_i}$$

**逻辑说明：**
1. 将每个顶点的坐标乘以其权重。
2. 累加所有加权坐标。
3. 除以总权重，得到最终的球心坐标 $(x, y, z)$。

### 第三步：计算近似半径
有了球心后，通过计算顶点到球心的距离来确定半径，。推荐使用**加权平均半径法**，它比取最大值更稳健，能平滑掉个别突出的顶点。
**数学公式：**
1. 计算单点距离：$d_i = \| p_i - C_{center} \|$
2. 计算加权平均半径：$R = \frac{\sum_{i=1}^{N} (d_i \times w_i)}{\sum_{i=1}^{N} w_i}$
### 第四步: 修改骨骼参数
根据计算得到的半径，修改骨骼参数。
*   **胸上2骨骼位置**：头部尾部位置：X Y与球心位置相同 Z位置为球心位置加减半径
*   **胸上骨骼位置**：头部位置： Z位置为球心位置加半径 X位置与原始骨骼相同 Y位置与原骨骼相同。 尾部位置： Z位置为球心位置加半径 X Y与球心位置相同。
*   **胸下骨骼位置**：头部位置：X Y与原始骨骼相同 Z位置为球心位置减半径。尾部位置： Z位置为球心位置减半径 X Y与球心位置相同。

### 第五步：设置刚体参数
根据计算得到的球心和半径，设置刚体参数。
*   **位置**：$C_{center}$
*   **半径**：$R$
